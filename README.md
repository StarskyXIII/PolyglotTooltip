# PolyglotTooltip

PolyglotTooltip is a client-side mod that adds extra translated name lines to tooltips and improves cross-language search in supported mod UIs.

## Features

- Show item names in additional languages directly in tooltips.
- Extend cross-language search in supported integrations such as JEI, EMI, AE2, RS2, and related mod UIs.
- Optionally treat Simplified and Traditional Chinese as interchangeable during search.

## Credits

PolyglotTooltip bundles [Opencc4j][opencc4j] for Simplified/Traditional Chinese conversion used by cross-search matching.
